早餐店線上點餐

歡迎註冊試用

Demo link: https://coldingpotato.github.io/onlineOrder/

## UI功能簡介
### 前台
> 可以使用demo專用帳號登入體驗→小明
- 登入、註冊功能
- 提供點餐服務
- 歷史訂單之查詢功能
### 後台
> 可以使用demo專用帳號登入體驗→阿姨
-  顯示當前訂單(未處理之訂單)
-  菜單管理(新增、編輯、停售)
-  營業狀況之圖表分析

## 技術簡介
### 前端套件
- jQuery
- Axios
- SweetAlert2
- c3.js
- GA4

### 後端模擬
- json-server-auth

### 伺服器架設
